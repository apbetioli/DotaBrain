# DotaBrain

App for learning and suggesting the best counter pick

## Install

```
npm install
```

## Depends

Depends on <a href="https://github.com/apbetioli/Dota2Mongo">Dota2Mongo</a> to populate the database.

## Prepare the data

```
node prepare.js
```

## Train the network

```
node train.js
```

## Run the server

```
node server.js
```
